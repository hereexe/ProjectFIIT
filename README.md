# BMI калькулятор (HTML/JS)

Простой одностраничный калькулятор индекса массы тела (BMI), работающий целиком в браузере без серверной части.

## Описание

Приложение содержит:

- форму для ввода **роста** (в сантиметрах) и **веса** (в килограммах);
- JavaScript‑скрипт, который вычисляет BMI по формуле  
  \(\text{BMI} = \text{weight} / (\text{height}/100)^2\);
- вывод числового значения BMI и текстовой категории (недостаток веса, норма, избыточный вес, ожирение);
- базовое оформление с помощью встроенного CSS.

## Требования

- Любой современный браузер (Chrome, Edge, Firefox, Safari и т.п.).
- Операционная система не важна — проект запускатся локально.

## Запуск локально

1. Скопируйте или скачайте файлы проекта (достаточно `index.html` и `README.md`) в любую папку на вашем компьютере.
2. Откройте папку с проектом.
3. Дважды щёлкните по файлу `index.html`, чтобы открыть его в браузере.  
   - Либо кликните правой кнопкой по `index.html` → **Открыть с помощью** → выберите браузер.

После открытия страницы в браузере:

1. Введите рост в поле «Рост» (в сантиметрах).
2. Введите вес в поле «Вес» (в килограммах).
3. Нажмите кнопку **«Рассчитать BMI»**.
4. Ниже формы появится рассчитанное значение BMI и соответствующая категория.

Проект не требует установки зависимостей, серверов или дополнительной настройки.
